autodoxygenindex Directive Example¶

Working Example¶

This should work:

.. autodoxygenindex:: class.h
   :source: class

With the following config value:

breathe_projects_source = {
     "class" : "../examples/doxygen"
     }

It produces this output:

class Test
#include “inc/class.h”

This is a test class.

Some details about the Test class

file class.h

dir /home/mike/root/projects/sphinx/breathe/examples/doxygen

dir /home/mike/root/projects/sphinx/breathe/examples
